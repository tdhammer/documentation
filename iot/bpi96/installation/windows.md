## 1) Connect the Debugger to the Board as Follows

| JLink Debugger | Bpi NB-IOT |
|:--------------:|:----------:|
| 3v3 [Pin 1]    | VCC        |
| GND [Pin 6]    | GND        |
| SWDIO [Pin 7]  | DIO        |
| SWCLK [Pin 9]  | CLK        |
